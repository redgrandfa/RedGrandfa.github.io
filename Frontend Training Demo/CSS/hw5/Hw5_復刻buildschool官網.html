協會的理念

「社團法人台灣青築未來協會 Build the Future 」已於 2020/6/19日向內政部完成立案正式成立了!

名字叫「青築未來」的意義就是 「幫青年人築起他的未來」 這是 Build School 校友會暨 NPO非營利組織，協會的目標除了幫助年輕人成為軟體工程師，回饋社會，同時也是 Build School 校友會，成為歷屆學員的交流平台

協會的遠景 - 希望每一位校友，能以身為 Build School 一份子感到驕傲! 除了校友在自己的工作領域持續積極表現、為你們的工作創造價值外，我們也能持續公益回饋，只要長期去做，我們對社會就會有更大的正面影響力，可以影響並幫助更多人..

Pay it Forward

讓愛傳出去

公益回饋免費培訓

幫助年輕人成為軟體工程師

Build School 目前在台北/新竹持續開設軟體工程師就業培訓班，每一期將由協會贊助一些弱勢的學員或提供公益免費名額，只要他/她動機強，想變得更好，改變她的職涯及人生，資助他們進入 Build School 培訓。結訓後順利找到工程師的工作有了薪水後，再回饋給協會，或是撥時間來當助教指導未來需要幫助的學員們。

協會本身也是 NPO 非營利之組織，這就是「把愛傳下去」Pay it forward 的概念，是一種很棒的型式，去幫助下一個需要幫助的人，一棒一棒傳下去，如此源源不絕....

期許 Build School 校友們 “Be a giver, you get more” - 持續去回饋，你能獲得更多!

Helping others actually

helps yourself

幫助非營利組織數位轉型

Empower NPOs for digital transformation

支持並贊助非營利組織 NPO 的資訊人員，參加 Build School 培訓或提供資訊技術協助，提升該組織之IT能量及軟體開發應用之效能。

我們曾贊助「失親兒福利基金會」「喜憨兒基金會」「伊甸基金會」 之 IT 人員參與培訓，期望協會能支持更多 NPO 組織，進而協助其數位轉型，擴大其影響力及價值，幫助他人發揮價值，就是幫助自己!

於2021/4月起於台北市就業服務處下的「臺北青年職涯發展中心Taipei Youth salon」持續舉辦「善用數位工具」的一系列課程。

校友連結

Alumni Network

從 Build School 結訓後進入軟體工程領域的學員愈來愈多，協會就是校友會，除了工作上可以互相交流外，更重要的是校友們能在軟體工程領域持續學習及成長。型式可以是讀書會、技術分享會、Meetups、或是代表公司前來軟體工程師徵才
